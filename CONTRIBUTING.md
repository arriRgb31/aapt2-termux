# Contributing

Thank you for your interest in contributing.

## Before Submitting

- Test changes on ARM64 Termux.
- Ensure the project builds successfully using `./build.sh`.
- Keep commits focused on a single feature or fix.
- Update documentation if needed.

## Pull Requests

Please include:

- A clear description of the change.
- Steps to reproduce (for bug fixes).
- Build or test results when applicable.

## Bug Reports

Include:

- Android version
- Device
- Termux version
- Steps to reproduce
- Terminal output or build log
